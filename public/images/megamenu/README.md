# Megamenu Images

This folder contains images displayed in the Home megamenu dropdown.

## Required Images

To match the Figma design exactly, replace the placeholder images with:

### 1. pants.png (or pants.jpg)
- **Content**: Fashion model wearing green pants or dress
- **Orientation**: Portrait (vertical)
- **Recommended size**: 170px × 200px (or higher resolution with same aspect ratio)
- **Format**: PNG or JPG
- **Description**: Should show a full-body or 3/4 shot of a model wearing the featured pants/clothing item

### 2. bags.png (or bags.jpg)
- **Content**: Fashion model wearing or carrying a brown leather bag
- **Orientation**: Portrait (vertical)
- **Recommended size**: 170px × 200px (or higher resolution with same aspect ratio)
- **Format**: PNG or JPG
- **Description**: Should show a model with the featured bag accessory

## How to Update

1. Export the images from your Figma design
2. Save them to this folder as `pants.png` and `bags.png` (or .jpg)
3. If using .jpg format, update the file extensions in `components/Navbar.tsx`
